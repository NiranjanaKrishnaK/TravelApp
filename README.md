## ✈️ Travel Planner App

A full-featured travel planning application that helps users organize trips, manage bookings, track expenses, and generate trip summaries — all in one place.

## Features Implemented

1. User Registration and Login 
   - Secure account creation and login system  
   - Personal profile management

2. Travel Booking Module  
   - Input destination, travel dates, and mode of transport  
   - Record hotel, flight, and other booking details
3. Itinerary Management  
   - Create, edit, and view day-wise itineraries  
   - Add activities and notes for each day

4. Budget Planning and Expense Tracking 
   - Estimate total trip cost  
   - Categorize expenses (accommodation, food, travel, etc.)  
   - Track spending in real-time

5. Trip Summary Generation
   - Auto-generate a summary report of the trip  
   - Includes itinerary and budget breakdown

6. Data Storage and Retrieval  
   - All user and trip data stored securely  
   - Easily retrievable for future reference

7. Search and Filter Options
   - Search saved trips  
   - Filter itineraries by date, location, or budget range
